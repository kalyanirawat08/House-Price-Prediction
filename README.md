# 🏠 House Price Prediction using Machine Learning

A simple **House Price Prediction** project built with **Python**, **Scikit-learn**, **NumPy**, and **Tkinter**. This application uses a **Linear Regression** model to estimate house prices based on user inputs through a graphical user interface (GUI).

## 🚀 Features

* Predicts house prices using Machine Learning.
* Simple and interactive Tkinter GUI.
* Takes the following inputs:

  * Area (sq ft)
  * House Age (years)
  * Number of Rooms
* Displays the estimated house price instantly.
* Handles invalid user inputs with error messages.

## 🛠️ Technologies Used

* Python
* Tkinter
* NumPy
* Scikit-learn (Linear Regression)

## 📂 Project Structure

* `house_price_prediction.py` – Main application containing the ML model and GUI.

## ▶️ How to Run

1. Clone this repository.
2. Install the required libraries:

   ```bash
   pip install numpy scikit-learn
   ```
3. Run the Python file:

   ```bash
   python house_price_prediction.py
   ```

## 📊 Machine Learning Model

The project uses a **Linear Regression** model trained on a small sample dataset containing:

* Area
* House Age
* Number of Rooms

The model predicts the estimated house price in lakhs based on the provided inputs.

## 📸 Output

The application provides a user-friendly interface where users can enter house details and instantly receive a predicted price.
